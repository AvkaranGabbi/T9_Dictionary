# T9_Dictionary
A Dictionary of words that include the option to search into those words and add new words.
A separate text file has been uploaded named as "words.txt", which include pre written english words in alphabetical order. In the cpp file, the person can search for a word from all the words, where on inputting each character, the corresponding words show up.
And if, the searched word is not present in the file beforehand, the user gets an option to add the new word into the word file, which too will get placed into the words at its alphabetic place.
